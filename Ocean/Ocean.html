Unity Web Player | Ocean Shader Experiments

This is an implementation of Jerry Tessendorf's paper "Simulating Ocean Water." It generates a random wave field, then animates it using a CPU based Fast Fourier Transform. Click on the compass to change the wind direction, and click the slider to change the wind intensity.
